# Homebrew Tap

Personal Homebrew tap for tools by Dominic Rodemer.

## Installation

```bash
brew tap domzilla/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [queuestack](https://github.com/domzilla/queuestack) | Minimal, scriptable task and issue tracker for agent-driven workflows |

## Installing a Formula

```bash
brew install domzilla/tap/queuestack
```

Or after tapping:

```bash
brew install queuestack
```

## Updating

```bash
brew update
brew upgrade queuestack
```

## Issues

If you encounter problems with any formula, please open an issue on the respective project repository.
