# Homebrew Tap

Personal Homebrew tap for tools by Dominic Rodemer.

## Installation

```bash
brew tap domzilla/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [qstack](https://github.com/domzilla/qstack) | Minimal, scriptable task and issue tracker for agent-driven workflows |

## Installing a Formula

```bash
brew install domzilla/tap/qstack
```

Or after tapping:

```bash
brew install qstack
```

## Updating

```bash
brew update
brew upgrade qstack
```

## Issues

If you encounter problems with any formula, please open an issue on the respective project repository.
